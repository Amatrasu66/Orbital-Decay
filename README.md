# Orbital Decay

A fast-paced orbital survival game built entirely with **HTML5 Canvas** and **Vanilla JavaScript**.

Pilot your satellite around a dying star, collect data probes, avoid deadly solar flares, and survive as the star slowly collapses inward. The closer you orbit, the faster you move and the greater the rewards.

## 🎮 Play Online

**Live Demo:** https://orbital-decay-three.vercel.app

No installation required. Play directly in your browser.

## 🎮 Controls

### Desktop

| Key           | Action        |
| ------------- | ------------- |
| W / ↑         | Boost outward |
| S / ↓         | Brake inward  |
| P / Esc       | Pause         |
| Enter / Space | Select        |

### Mobile

* Left side of the screen → Brake inward
* Right side of the screen → Boost outward
* Touch UI support
