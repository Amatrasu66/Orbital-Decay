# Orbital Decay

Pilot your satellite around a dying star. Collect data probes, dodge solar flares, and survive as the star collapses inward.

## 🎮 Play Online

**Live Demo:** https://orbital-decay-three.vercel.app

No installation required. Play directly in your browser.

## Features

- Kepler-inspired orbital mechanics
- 6 game modes
- Wave system with random modifiers
- Power-ups and upgrades
- XP rank progression
- 20 achievements
- Daily missions
- Unlockable ship and star skins
- Procedural audio
- Touch and keyboard controls
- Persistent save system
